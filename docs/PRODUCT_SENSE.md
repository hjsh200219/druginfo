# DrugInfo MCP - Product Sense

## 제품 개요

DrugInfo MCP Server는 EDB(이디비) 의약품 정보 시스템의 데이터를 MCP 프로토콜을 통해 AI 에이전트에 제공하는 브릿지 서버입니다.

## 핵심 사용자

### Primary: AI 에이전트 (Claude Desktop, IDE Agent)
- 의약품 정보를 자연어 질의로 검색
- 동일 성분 의약품 비교
- 생물학적동등성(생동PK) 정보 확인

### Secondary: 개발자
- MCP 도구를 통한 의약품 데이터 접근
- CLI를 통한 인증 테스트

## 핵심 사용 시나리오

### 1. 의약품 정보 조회
```
사용자: "타이레놀 정보 알려줘"
-> druginfo_list_product(pillName="타이레놀") -> ProductCode 추출
-> druginfo_get_product_by_code(code=...) -> 상세 정보 제공
```

### 2. 동일 성분 의약품 검색
```
사용자: "실데나필 성분의 다른 약 찾아줘"
-> druginfo_list_product_edicode_same_ingredient(MasterIngredientCode=...)
-> korange.생동PK 필터링으로 생동성 확인된 제품 선별
```

### 3. 의약품 코드 해석
```
사용자: "E201207542ATB7D 코드 해석해줘"
-> ProductCode 구조 분석: 전문의약품, 품목기준코드 201207542, 투여경로+제형 ATB
```

## 제품 원칙

1. **투명한 인증**: 사용자가 토큰 만료를 의식하지 않도록 자동 재인증
2. **토큰 절약**: 응답 압축 + 전용 도구 우선 사용으로 LLM 토큰 비용 최소화
3. **안전한 실패**: API 변경 시에도 원본 데이터 반환으로 서비스 연속성 보장
4. **도메인 정확성**: 의약품 코드 체계와 생동성 정보의 정확한 전달

## 의약품 코드 체계 요약

| 코드 | 자릿수 | 발급기관 | 특징 |
|---|---|---|---|
| ProductCode | 15자리 | 터울 자체 | 고유, 변경 없음 |
| 주성분코드 | 9자리 | 심평원 | 동일 성분 공유 |
| EDI(보험)코드 | 9자리 | 심평원 | 변경 가능 |
| 표준코드 | 13자리 | 심평원 | 변경 가능 |
| 품목기준코드 | 9자리 | 식약처 | 변경 없음 |
| ATC코드 | 7자리 | WHO | 국제 표준 |
